# Frontend Layer

This folder is reserved for frontend client implementations.

## Current status
The existing Flutter application currently lives at repository root.

## Structure
- `mobile_flutter/` target home for Flutter mobile client
- `web_app/` optional web frontend
- `admin_dashboard/` optional operations/admin frontend
- `shared/` shared design tokens, assets, and UI contracts

## Next implementation steps
1. Keep building the current Flutter app in root for now.
2. When ready, move root Flutter project into `frontend/mobile_flutter/`.
3. Add separate package management for web/admin frontends if needed.
